# I Loan Management System

A full-stack loan management application developed to simplify and automate the loan processing workflow for financial institutions.

## Project Overview

The I Loan Management System helps manage the complete loan lifecycle, including customer registration, loan enquiries, application processing, document management, approvals, reporting, and user access control.

## Features

- User Authentication & Authorization
- Role-Based Access Control (RBAC)
- Customer Management
- Loan Enquiry Management
- Loan Application Processing
- Document Upload & Verification
- Loan Approval Workflow
- Reports & Analytics Dashboard
- Email Notifications
- Secure Data Management

## Technology Stack

### Frontend
- React.js


### Backend
- Node.js


### Database
- MySQL

## Purpose

This project was developed to digitize and streamline the loan management process by reducing manual work, improving data accuracy, and providing a centralized platform for managing customers, loan applications, approvals, and reports.

### Backend
